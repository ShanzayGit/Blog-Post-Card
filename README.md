# 📝 Blog Post Card

A responsive **Blog Post Card UI** built using **HTML and CSS**.

This project displays a travel-themed blog preview card featuring an image, title, short description, and a call-to-action button.

## 📌 Features

* Clean card-based layout
* Responsive centered design
* Styled image section
* Hover effect on action button
* Rounded container with modern appearance
* Internal navigation using links

## 🛠 Technologies Used

* HTML5
* CSS3

1. Download or clone the project.
2. Open the project folder.
3. Ensure the image folder is included.
4. Open `index.html` in your browser.

## 🎨 Design Overview

### Layout

* A centered blog card container

### Components

* **Title** → Blog category heading
* **Excerpt** → Short blog description
